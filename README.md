## setup


前端：
`cd client && cd $_;npm i`
`npm run start`

后端:
`cd server && cd $_;npm i`
`npm run start`

后端调试
`npm run inspect`
`debugger`设置断点
`repl`打印变量
`cont, c`: Continue execution
`next, n`: Step next
`step, s`: Step in
`out, o`: Step out
`restart`: Restart script
